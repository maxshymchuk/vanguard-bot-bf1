VG Bot Setup

Requirements for this build

  • Game must be in English
    • We should be able to make versions for most other languages in the future if required
  • If you are running the exe file you will (probably) need to run the bot in fullscreen mode
    • This is due to some strange anti-screenshot functionality BF1 seems to have in borderless/windowed mode, at least on my PC
    • If you want to run it in borderless or windowed mode ping us and we can send you the Python version which works in borderless and windowed

Setup steps

Video: https://youtu.be/kCFOP2NYkRQ

  1. Launch BF1
  2. Video settings → Set to borderless
  3. Go into spectate
  4. Go to third person view → Options → Show player card on
  5. Launch vgbot.exe
  6. An overlay will appear, follow the setup instructions
  7. Once completed the overlay will disappear
  8. Video settings → Set to fullscreen
  9. Go back to the program and press Y to test the config
  10. Wait a few seconds for the bot to cycle three players
  11. Go to the folder with the exe, open screenshots folder, check the images look correct
  12. Go back to the program and press enter. The bot will now run

Config File

You shouldn't need to change much in the config json file. However here is a description of some things you may want to change

  • Pause hotkey: Press this to toggle the bot cycling players. By default it is Ctrl + P
  • Rotation threads: More threads will mean the bot can run faster but uses more of your CPU. The default, 0, uses the maximum number of your cores

Important: If you do not use the default colour values in BF1 (red for enemy, blue for team, green for squad) you will need to change the colour option in the config file.
Go to colorblind option and write in your custom values as they are written (ally_color = team color)

Command Line Arguments

There are some command line arguments you can use.

  • Right click anywhere in the folder and click "Open in Terminal"
  • Write .\vgbot.exe --help to view all options
  • Use as many of these options as you want e.g .\vgbot.exe -s -sc to make the bot run on launch and save screenshots of kicked players

You can have the bot always use one of these options every time you run it.

  • Right click on vgbot.exe
  • Show more options → Create shortcut
  • Right click on the shortcut → Properties
  • In the target field, after the text already there, enter the argument you want the shortcut to use
    • e.g add -s for the bot to run on launch
